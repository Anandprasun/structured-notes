# Examples

Real input → output pairs produced by the skill. Each folder has the raw
`input.txt` and the resulting `notes.md`. Read past the `— — —` divider in each
output to see the **audit** and **verification** — the part a plain summary skips.

| Example | Input type | What it shows |
|---------|-----------|---------------|
| [`meeting-finance/`](./meeting-finance) | Meeting (cost-cutting) | The audit catches that the proposed cuts (240k) fall **60k short** of the 300k mandate, and that an exemption makes it worse — arithmetic a plain summary misses. |
| [`transcript-churn/`](./transcript-churn) | Call transcript (churn save) | Theme clustering, verbatim quotes with timestamps, and the buried deadline (renewal auto-charges on the 20th) surfaced as a risk. |
| [`brain-dump/`](./brain-dump) | Personal brain-dump | Light structure over chaos — intent split (Do / Ideas / Open loops), voice preserved, deadlines flagged, nothing dropped. |
